# maven2

Add this repository in the POM.xml

```xml
<repositories>
  <repository>
    <id>my-github</id>
    <url>https://maven.pkg.github.com/api2doc/maven2</url>
    <snapshots>
      <enabled>true</enabled>
    </snapshots>
    <releases>
      <enabled>true</enabled>
    </releases>  
  </repository>
</repositories>
```
